## Snake-water-gun-game
This is a simple Python implementation of the classic Snake, Water, Gun game. The user selects one of the three options, and the computer randomly selects its own choice. The game then determines the winner based on predefined rules.

# How the Game Works

Snake beats Water

Water beats Gun

Gun beats Snake

If both player and computer choose the same option, the result is a tie

# Features

Clean and simple command-line gameplay

Randomized computer choices

Input validation for user entries

Clear win, lose, and tie messages

# How to Play

Run the Python script.

Enter your choice:

s for Snake

w for Water

g for Gun

The computer will make its move automatically.

The game will display who wins.
